# 如何设置 GitHub Profile README

这个文件包含了您的 GitHub 个人主页的极客风格 README。

## 📝 设置步骤

要在您的 GitHub 个人主页（https://github.com/TingdeLiu）上显示这个极客风格的 README，请按照以下步骤操作：

### 1. 创建特殊的 Repository

GitHub 允许您创建一个与用户名同名的特殊 repository 来定制您的个人主页：

1. 访问 https://github.com/new
2. **Repository name** 填写：`TingdeLiu`（必须与您的用户名完全相同）
3. 选择 **Public**（必须是公开的）
4. 勾选 **Add a README file**
5. 点击 **Create repository**

### 2. 添加 README 内容

创建 repository 后：

1. 编辑 `README.md` 文件
2. 将 `GITHUB_PROFILE_README.md` 的内容完整复制到 `README.md`
3. 提交更改

### 3. 查看效果

- 访问 https://github.com/TingdeLiu
- 您应该能看到新的极客风格个人主页！
