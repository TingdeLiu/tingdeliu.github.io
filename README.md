## 参与贡献

每篇文章末尾都有"社区反馈卡片"，点击即可一键提交（自动带上结构化模板）：

- 🐛 **报告错误**：发现内容错误或笔误
- 📄 **推荐论文**：推荐值得收录的相关论文
- ✏️ **修改建议**：对内容组织、表述的改进建议

所有提交过反馈的朋友都会出现在文章末尾的 **社区贡献者墙** 上。也可以直接 [新建 Issue](https://github.com/TingdeLiu/Tingde.Liu.github.io/issues/new)。
